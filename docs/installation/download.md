# Download the code
Clone our repo from Github to your PC (`.\Paymenter`):

```jsx title="Windows"
git clone https://github.com/Paymenter/Paymenter
cd Paymenter
```
```jsx title="Linux"
$ git clone https://github.com/Paymenter/Paymenter
$ cd Paymenter
```
